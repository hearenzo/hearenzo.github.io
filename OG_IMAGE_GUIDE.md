# OG Image Design Guide

## 작업 특성 분석

작업의 핵심 개념:
- **프리즘**: 빛을 굴절시켜 본질을 드러내는 개념
- **사운드의 물질성**: 물리적 객체를 통한 사운드 탐구
- **기계적 리듬**: DC 모터, 피아노 와이어 등 기계적 요소
- **진동과 떨림**: 공간 전체로 퍼지는 진동의 질감
- **시간성의 해체**: 시간적 구조를 넘어선 공간적 경험

## 디자인 제안

### 옵션 1: 미니멀 프리즘 (현재 og-preview.svg)
- 프리즘 형태의 기하학적 요소
- 사운드 파형 시각화
- 미니멀한 타이포그래피
- 료지 이케다 스타일과 조화

### 옵션 2: 기계적 요소 중심
- 피아노 와이어, 모터 등 기계적 요소의 추상화
- 선과 원의 조합
- 진동을 나타내는 파형

### 옵션 3: 공간과 진동
- 공간을 채우는 진동 패턴
- 점과 선의 조합
- 미니멀한 색상 (흑백)

## 이미지 생성 도구 추천

1. **Figma** - SVG로 직접 디자인 가능
2. **Canva** - OG 이미지 템플릿 제공
3. **Adobe Express** - 빠른 이미지 생성
4. **Midjourney/DALL-E** - AI 이미지 생성 (프롬프트 제공 가능)

## 권장 사양

- 크기: 1200 x 630px (1.91:1 비율)
- 형식: PNG 또는 JPG
- 파일 크기: 1MB 이하
- 색상: 흑백 또는 미니멀한 색상

## 사용 방법

생성한 이미지를 `/assets/img/` 폴더에 저장하고, 각 페이지의 front matter에 추가:

```yaml
---
og_image: "https://hearenzo.github.io/assets/img/your-image.jpg"
---
```
